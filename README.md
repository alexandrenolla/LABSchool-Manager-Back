## Running the project


1. Clone the repository:

```bash
$ git clone https://github.com/alexandrenolla/LABSchool-Manager-Back
$ cd LabSchoolAPI
```

2. Install the dependencies by running the command:

```
 dotnet build
```

3. Run the command:

```
 dotnet run
```

The API will be available at http://localhost:5127/swagger.
